Concepts & Thinking

A brief overview of the mechanical principles I focus on.

⚡
Reactivity

Explored how change propagation works through observers and dependency graphs. Learned the practical tradeoffs between implementation simplicity and flexibility.

🔄
State Management

Investigated mutable vs immutable approaches. Implemented the command pattern to enable reliable undo/redo, and used batching to reduce computational overhead.

📦
Command Pattern

Encapsulating mutations into command objects provides a clean, auditable history of state. It transforms state management from a black box into a traceable sequence of events.

🐛
Debugging

Instrumenting state and commands so systems are auditable and explainable. The best learning tool is an observable system.
